Mobile Price Classification

In the ever-expanding mobile phone market, users face an increasing challenge: the quality and value of identity among the threshold models. Further complicating the problem is the lack of tools that facilitate effective comparison of manufacturer specifications and prices.

This project aims to close this critical information gap while developing a mobile pricing system. This system will use machine learning to analyze the key features and specifications of mobile phones. Analyzing this data very carefully, the system aims to establish a firm and accurate connection between the phone parts and the associated price ranges.
